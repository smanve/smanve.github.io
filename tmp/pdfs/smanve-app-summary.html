smanve.github.io

A personal portfolio and writing site for Manvendra Singh, built with Next.js and local MDX content. The app presents a landing page, project showcase, and blog-style long-form pages backed directly by files in the repo.

Who It Is For

Primary persona (inferred from repo): recruiters, hiring managers, and technical peers reviewing Manvendra Singh's background, projects, and engineering writing.

What It Does

  • Shows a personal intro, location, and external profile links on the home page.
  • Uses a sticky top nav for Home, Blog, and Projects routes.
  • Highlights the latest three projects on the landing page.
  • Publishes blog posts from local content/posts/*.mdx files.
  • Publishes project detail pages with hero images, tags, and GitHub/live links.
  • Adds a particle animation layer for the site's visual identity.

How To Run

  1. Install Node.js >= 18.17.
  2. From the repo root, run npm install.
  3. Start the app with npm run dev and open http://localhost:3000.
  4. Environment variables or extra setup: Not found in repo.

How It Works

1
Routes in src/app render the Home, Blog, and Projects pages inside a shared layout and sticky header.
2
Server-side helpers in src/lib/posts.ts and src/lib/projects.ts read local MDX files with fs and gray-matter to build content lists.
3
Dynamic [slug] routes call generateStaticParams and compile MDX with next-mdx-remote/rsc, remark-gfm, and rehype-slug.
4
Assets come from public/, while client-only components handle active nav state and the react-particles background driven by particles.json.

Backend API, database, authentication service, and external data integrations: Not found in repo.
